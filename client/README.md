# HealthSnap - Client (Frontend)

HealthSnap is a **React (Vite) PWA** that allows users to scan barcodes and get detailed food product insights.

## 🚀 Features  
- 📸 **Barcode Scanning** - Scan food product barcodes.
- 🔎 **AI-Powered Analysis** - Get ingredient insights.
- 🛒 **Nutri Cart** - Store and compare products.
- 📱 **PWA Support** - Works offline like a native app.
