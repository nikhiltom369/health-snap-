
# HealthSnap - Backend (API)

The backend of HealthSnap is built using **Node.js (Express)** and provides authentication, product details, and AI-powered insights.

## 🚀 Features  
- 🔐 **JWT Authentication** - Secure user login & registration.
- 🏷️ **Product API** - Fetch product details via barcode.
- 🤖 **AI Integration** - Get detailed ingredient analysis.
