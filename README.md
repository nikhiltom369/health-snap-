# HealthSnap

HealthSnap is a mobile-friendly web application built using the **MERN stack** (MongoDB, Express, React, Node.js). It allows users to scan barcodes on packaged food products and retrieve detailed product information, including nutritional insights, ingredient analysis, and personalized consumption recommendations powered by AI.

## Features

- 📸 **Barcode Scanner** - Scan food product barcodes to retrieve detailed information.
- 🔎 **Product Insights** - View nutritional facts, ingredients, and allergen details.
- 🤖 **AI-Powered Analysis** - Get personalized recommendations on food suitability.
- 📋 **Nutri Cart** - Add scanned products to the Nutri Cart for later review.
- ⚖️ **Compare Products** - Select items from the Nutri Cart to compare nutritional values.
- 🔐 **Authentication** - Secure login using JWT authentication.

## Tech Stack

- **Frontend:** React (Vite), Recharts (for graphs)
- **Backend:** Node.js, Express, MongoDB
- **Authentication:** JWT
- **Barcode Scanning:** `react-qr-barcode-scanner`, `zxing-js/library`
- **AI Integration:** OpenAI Gemini API (for ingredient analysis)
- **Hosting:**
  - Frontend: Vercel
  - Backend: Render
